# Retail Orders Data Analysis

## Overview

This is a beginner-level data analysis project built using the Retail Orders dataset from Kaggle.

The primary goal of this project is to refresh and strengthen my data analysis skills by working with a real-world dataset using Python, Pandas, and SQL. The project covers data cleaning, data loading, and solving business-related analytical questions using SQL.

## Dataset

**Source:** Retail Orders Dataset (Kaggle)

https://www.kaggle.com/datasets/ankitbansal06/retail-orders

## Tech Stack

- Python
- Pandas
- Jupyter Notebook
- MySQL
- SQL
- Git & GitHub

## Project Workflow

1. Downloaded the dataset using the Kaggle API.
2. Loaded the dataset into a Pandas DataFrame.
3. Performed data cleaning and preprocessing.
4. Loaded the cleaned data into MySQL.
5. Solved business-related analytical questions using SQL queries.

## Data Cleaning Performed

- Handled missing values
- Standardized column names
- Converted data types where required
- Prepared dataset for SQL analysis

## Business Questions Solved

### 1. Top 10 Highest Revenue Generating Products
Identified the products generating the highest sales revenue.

### 2. Top 5 Highest Selling Products in Each Region
Ranked products within each region based on total sales.

### 3. Monthly Sales Comparison (2022 vs 2023)
Compared month-wise sales performance across two years.

### 4. Highest Selling Month for Each Category
Determined the best-performing month for every product category.

### 5. Sub-Category with Highest Sales Growth
Calculated year-over-year sales growth and identified the fastest-growing sub-category.
